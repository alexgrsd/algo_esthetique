# Atelier Esthétique et algorithmique

## Objectif

Le principe de cet atelier est de réaliser 5 mini projets, tous avec une contrainte sur soit l'outil a utiliser, soit sur une technique d'algo (fractale, automate cellulaire, etc)

Le projet qui me plait le plus reste le 5eme (et par extensionn le 4eme), car représentant vraiment ce que j'aime créer, mais en réalisant l'algo derrière. Ce dit projet est aussi un projet personnel, que je continuerai moi-même.
